---
name: Find Images
description: Locate images from past posts that could be reused for a new piece.
---

Use this skill when the user needs an image for a new post and may be able to reuse an existing one rather than creating something new.

**Steps**
1. Ask the user what concept, topic, or visual idea the image should represent.
2. Search existing posts for images tagged with related concepts.
3. Return a list of candidate images with: the post they came from, a description, and the file path or URL.
4. Recommend the best match and explain why it fits.
5. If no suitable image exists, say so clearly so the user knows a new one is needed.
