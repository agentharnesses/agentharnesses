name: Draft Post
description: Draft a new post from an outline, brief, or topic description.
---

Use this skill when the user wants to write a new piece from scratch or from an outline.

**Steps**
1. If the user provides only a topic, ask for: target audience, desired length, and key points to cover.
2. Check `references/topic-index.md` to see if the topic overlaps with existing posts — surface any relevant prior work.
3. Follow all rules in `references/style-guide.md` for tone, structure, and formatting.
4. Draft the post. Use headers to organize long pieces. Include a clear introduction and a conclusion.
5. After drafting, list any images that might strengthen the piece and suggest running `find-images` to locate them.
