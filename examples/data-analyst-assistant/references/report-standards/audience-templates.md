# Audience Templates

## Executive Summary

Intended for: C-suite, board, non-technical stakeholders.

Structure:
1. One sentence stating the headline finding
2. Three bullet points of supporting evidence with numbers
3. One sentence on recommended action or next step

Length: under 100 words.

## Team Lead Digest

Intended for: team managers reviewing weekly metrics.

Structure:
1. Period and scope (e.g. "Week of 2024-03-04, North America")
2. Key metrics table (metric, current period, prior period, change %)
3. Notable anomalies or outliers, if any

Length: one page maximum.

## Ad-hoc Data Pull

Intended for: internal analysts or engineers requesting raw data.

Structure:
1. Query description and parameters used
2. Row count and column list
3. Raw markdown table (truncated to 50 rows if larger; note full count)
