# Reporting Standards

## Numbers

- Round currency to 2 decimal places and prefix with `$`
- Round percentages to 1 decimal place and suffix with `%`
- Use commas as thousands separators (e.g. `1,234,567`)

## Tables

- Always include a header row
- Left-align text columns; right-align numeric columns
- Sort by the most meaningful column (usually date descending or value descending)

## Charts

- Line charts for trends over time
- Bar charts for comparisons across categories or cohorts
- Avoid pie charts

## Summaries

- Lead with the single most important finding
- Use bullet points for supporting detail
- Keep summaries under 150 words unless the user asks for more depth
